# Sprint 3

## Objectifs
- Compléter la gestion des projets (modification, suppression, visualisation, gestion des membres)
- Implémenter la gestion complète des user stories et des tâches
- Mettre en place la gestion des sprints
- Développer les fonctionnalités de visualisation avancées
- Implémenter la gestion des versions
- Mettre en place les tests unitaires pour les nouvelles fonctionnalités

## Livrables du sprint 3
- Interface complète de gestion des projets (CRUD complet)
- Système de gestion des user stories et tâches opérationnel
- Gestion des sprints fonctionnelle (création, démarrage, terminaison)
- Fonctionnalités avancées du Kanban (préférences, statistiques, permissions)
- Gestion des versions de projet
- Suite de tests unitaires couvrant les nouvelles fonctionnalités
