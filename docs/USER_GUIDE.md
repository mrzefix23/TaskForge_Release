## Inscription et Connexion
Pour commencer à utiliser TaskForge, vous devez d'abord créer un compte :
1. Cliquez sur le bouton "Créer un compte" sur la page d'accueil.
2. Remplissez le formulaire d'inscription avec vos informations personnelles. (Nom d'utilisateur, Adresse email, Mot de passe)
3. Validez en cliquant sur "Créer mon compte".
4. Une fois créé, cliquez sur "Se connecter" et entrez vos identifiants pour accéder à votre tableau de bord.

Si vous avez déjà un compte, appuyez simplement sur "Connexion" et entrez vos informations.

## Création et Gestion des Projets
Pour créer un nouveau projet dans TaskForge, suivez ces étapes :
1. Depuis votre tableau de bord, cliquez sur "Voir mes projets" pour accéder à la liste de vos projets.
2. Pour créer un nouveau projet, cliquez sur "Créer un projet".
3. Une fois sur le formulaire de création, entrez le nom du projet, une description et ajoutez des membres si nécessaire.
4. Cliquez sur "Créer le projet" pour finaliser.

Une fois le projet créé, vous pouvez :
- Supprimer le projet en cliquant sur l'icône de la corbeille.
- Modifier les détails du projet en cliquant sur l'icône de crayon.

Si vous décidez de modifier un projet, plusieurs options s'offrent à vous :
- Changer le nom du projet.
- Mettre à jour la description.
- Ajouter ou supprimer des membres.

## Gestion des Tâches
Pour accéder à la gestion des tâches :
1. Sélectionnez un projet depuis votre tableau de bord.
2. Cliquez sur le projet pour ouvrir sa vue détaillée.
