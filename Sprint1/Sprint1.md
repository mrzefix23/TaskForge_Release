# Sprint 1

## Objectifs
- Créer l'environnement de développement
- Mettre en place les outils nécessaires pour le développement collaboratif

- Base de données des utilisateurs créée
- Formulaire d’enregistrement fonctionnel
- Formulaire de connexion fonctionnel
- Documentation technique initiale
